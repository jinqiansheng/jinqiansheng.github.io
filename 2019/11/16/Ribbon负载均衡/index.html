Ribbon负载均衡

Nov 16, 2019点击

Spring Cloud Ribbon是基于Netflix Ribbon实现的一套客户端负载均衡工具。
主要提供客户端的软件负载均衡算法，将Netflix的中间层服务连接在一起，提供一系列完善的配置项，如连接超时、重试等。
简单来说就是在配置文件中列出Load Balancer(简称LB)后面所有的极其，Ribbon自动帮你基于某种规则去连接，并可以自定义负载均衡算法。
